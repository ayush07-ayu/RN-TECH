# RN-Tech Website

RN-Tech is a modern and responsive multi-page business website developed using HTML, CSS, and JavaScript. The website features a glowing dark red and black theme, interactive UI, background visuals, and a professional design.

## Features
- Responsive Design
- Modern UI/UX
- WhatsApp Chat Button
- Contact Page
- Course Page
- Landing Page
- Background Image/Video Integration

## Tech Stack
- HTML5
- CSS3
- JavaScript
